I've worked on a variety of projects over the years, ranging from web applications to low-level systems programming. Here are a few examples of my recent work:

### 1. SvelteChat - Svelte/Tailwind/SurrealDB
![Svelte Chat](/image/svelte-chat.png)
SvelteChat is a real-time chat application that I built using Svelte, Tailwind and SurrealDB. The application is built with Svelte, a lightweight and modern JavaScript framework, which provides a reactive and easy-to-use interface for the user. It leverages SurrealDB, a scalable and reliable Backend-as-a-Service (BaaS), for storing and retrieving chat messages in real-time. The application allows users to create chat rooms, join existing ones, and send private messages to each other in real-time.

### 2. Chirper - SvelteKit/Tailwind/SurrealDB
![Chirper](/image/chirper-profile.png)
A microblogging site like twitter made with SvelteKit/Tailwind/SurrealDB. It's fully functional. Featues include user authentication and authorization, create public/private chirps (posts), like/unlike chirps, comment on chirps, follow/unfollow users, join communities, post on communities, simple recommendation algorithm based on user's recent interactions. I also used docker to deploy the app.
