### 1. ChatApp - Svelte/Pocketbase/Go
![Svelte Chat](/image/svelte-chat.png)
ChatApp is a real-time chat application that I built using Svelte, Pocketbase, and Go. The frontend is built with Svelte, a lightweight and modern JavaScript framework, which provides a reactive and easy-to-use interface for the user. The backend is built with Go, a fast and efficient programming language, on top of Pocketbase, a self-hosted BAAS which can be extended as needed. It can store and retrieve chat messages in real-time. The application allows users to create chat rooms, join existing ones, and send messages to each other in real-time.
