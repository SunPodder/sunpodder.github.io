I've worked on a variety of projects over the years, ranging from web applications to low-level systems programming. Here are a few examples of my recent work:

### 1. [CodeBin](https://github.com/SunPodder/codebin) - Python/Flask/React

This project is a code sharing site similar to Pastebin, where users can share snippets of code with others. I built the backend with Python and Flask, which provides a simple and flexible framework for building web applications. For the frontend, I used React to create a modern and responsive user interface. Some of the features of the site include syntax highlighting, password protection for private snippets, and the ability to customize the expiration time for snippets.


### 2. [Blog Site](https://github.com/SunPodder/linuxian) - Django

I built this project using Django, a popular Python-based web framework. The site is a simple blog where users can create and publish articles, as well as view articles from other authors. I used Django's built-in authentication and authorization features to ensure that only authorized users can create or modify articles. The site is fully responsive and optimized for mobile devices.


### 3. [Simple HTTP Server](https://github.com/SunPodder/c-http-server) - C

For this project, I wanted to explore low-level systems programming by building a simple HTTP server from scratch. I wrote the server in C, which allowed me to work closely with the underlying operating system and network stack. The server is capable of serving static files, handling multiple client connections simultaneously, and responding to HTTP requests with custom headers and content.
